8 kyu: Write a function findNeedle() that takes an array full of junk but containing one "needle".

After your function finds the needle, it should return a message (as a string) that says: "found the needle at position " plus the index it found the needle.
Example: ["hay", "junk", "hay", "hay", "moreJunk", "needle"] --> "found the needle at position 5" 


def find_needle(haystack):
    for index, item in enumerate(haystack):
        if item == 'needle':
            return f'found the needle at position {index}'

DNA strand is never empty or there is no DNA at all (again, except for Haskell).
Example: "ATTGC" --> "TAACG" and "GTAT" --> "CATA"
